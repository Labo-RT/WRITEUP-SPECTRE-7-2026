# Question joueur

- 1/

Quelle est l'adresse IP du serveur principal ghostshark.local ?

- 2/

Il existe un sous-domaine n3t avec un TXT record contenant un secret. Trouvez-le !

- 3/ 

L'étape précédente a mentionnée un SERVICE ssh est utilisé. Trouvez le numéro de port utilisé.

- 4/

L’adresse IP du service SSH semble associée à un autre nom d’hôte via le reverse DNS. Identifiez ce nom et analysez les informations qu’il expose. Le flag est la version du service ssh du serveur (en entier).
